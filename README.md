# MediVault

MediVault is a secure cloud-based medical appointment and records management system developed for the **Database & Cloud Security (TCS 3651)** course.

## Features
- Patient and Admin authentication
- Secure appointment booking
- Role-Based Access Control (RBAC)
- Password hashing and validation
- SQL injection prevention
- Logging and audit trails
- Account lockout mechanism
- Database backup

## Technologies Used
- Streamlit
- SQLite3
- Python

## How to Run
```bash
pip install -r requirements.txt
streamlit run app.py
```

## Security Compliance
- STRIDE Threat Modeling
- PDPA 2010 Compliance
